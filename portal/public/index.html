Deploy in minutes

Install on your server, then complete first-run setup at /mantis/setup.

📦 Clone & run

git clone https://github.com/dev-gauravd/mantis.git
cd mantis
npm run install-all
npm run dev

🌐 First-run setup

Open your instance and create your local admin (not this portal login):

http://localhost:3000/mantis/setup

🔑 License key

Register here, get your license key, paste it in the Mantis setup wizard — or fetch it online during setup.
